## What is Mongoose?

**Mongoose** is an **ODM (Object Data Modeling) library** for **MongoDB** used in **Node.js** applications.

It provides a **structured way to model application data**, enforce **schemas**, perform **validation**, and interact with MongoDB using **JavaScript objects instead of raw queries**.

In simple words:

> Mongoose sits between **Node.js and MongoDB** and makes database operations easier and more organized.

---

# 1️⃣ Why Mongoose is Used

MongoDB itself is **schema-less**, meaning documents can have any structure.

Example without schema:

```json
{ "name": "Suman", "age": 22 }
{ "name": "Ram", "course": "JS" }
{ "age": 30 }
```

This can create **inconsistent data**.

Mongoose solves this by providing:

* Schema definition
* Data validation
* Middleware (hooks)
* Query helpers
* Easy CRUD operations
* Relationships between collections

# 2️⃣ Installing Mongoose

Install using npm:

```bash
npm install mongoose
```

---

# 3️⃣ Connecting to MongoDB

Example connection:

```javascript
import mongoose from "mongoose";

mongoose.connect("mongodb://127.0.0.1:27017/courses");

const db = mongoose.connection;

db.on("error", console.error.bind(console, "connection error:"));

db.once("open", () => {
  console.log("Connected to MongoDB");
});
```

Now Node.js can interact with the MongoDB database.

---

# 4️⃣ Schema in Mongoose

A **Schema** defines the **structure of documents**.

Example:

```javascript
const courseSchema = new mongoose.Schema({
  name: String,
  price: Number,
  assignments: Number,
  projects: Number
});
```

This means every document should follow this structure.

Example document:

```json
{
 "name": "Python for Beginners",
 "price": 100,
 "assignments": 12,
 "projects": 3
}
```

---

# 5️⃣ Model in Mongoose

A **Model** is a wrapper for the schema and represents a **collection**.

```javascript
const Course = mongoose.model("Course", courseSchema);
```

Now **Course model → courses collection in MongoDB**

---

# 6️⃣ Creating Documents (Create)

```javascript
const course = new Course({
  name: "NodeJS Basics",
  price: 200,
  assignments: 10,
  projects: 4
});

await course.save();
```

Or directly:

```javascript
await Course.create({
  name: "React Course",
  price: 300,
  assignments: 8,
  projects: 2
});
```

---

# 7️⃣ Reading Documents (Read)

Find all documents:

```javascript
const courses = await Course.find();
```

Find specific course:

```javascript
const course = await Course.findOne({ name: "React Course" });
```

Find by ID:

```javascript
const course = await Course.findById("65f123abc456");
```

---

# 8️⃣ Updating Documents (Update)

Update one document:

```javascript
await Course.updateOne(
  { name: "React Course" },
  { $set: { price: 350 } }
);
```

Update many documents:

```javascript
await Course.updateMany(
  { price: 0 },
  { $set: { price: 100 } }
);
```

Using find and update:

```javascript
await Course.findOneAndUpdate(
  { name: "NodeJS Basics" },
  { price: 250 }
);
```

---

# 9️⃣ Deleting Documents (Delete)

Delete one document:

```javascript
await Course.deleteOne({ name: "React Course" });
```

Delete many documents:

```javascript
await Course.deleteMany({ price: 0 });
```

# 🔟 Schema Validation

Mongoose can enforce validation rules.

Example:

```javascript
const courseSchema = new mongoose.Schema({
  name: {
    type: String,
    required: true
  },
  price: {
    type: Number,
    min: 0
  },
  assignments: Number
});
```

Now if name is missing:

```javascript
await Course.create({ price: 100 });
```

❌ Error occurs because **name is required**.

---

# 1️⃣1️⃣ Default Values

You can set default values.

```javascript
price: {
  type: Number,
  default: 0
}
```

If price not provided → automatically **0**.

---

# 1️⃣2️⃣ Middleware (Hooks)

Middleware runs **before or after operations**.

Example:

```javascript
courseSchema.pre("save", function(next) {
  console.log("Document is about to be saved");
  next();
});
```

Runs **before saving a document**.

---

# 1️⃣3️⃣ Relationships (Population)

MongoDB collections can reference other collections.

Example:

User schema

```javascript
const userSchema = new mongoose.Schema({
  name: String
});
```

Course schema

```javascript
const courseSchema = new mongoose.Schema({
  name: String,
  instructor: {
    type: mongoose.Schema.Types.ObjectId,
    ref: "User"
  }
});
```

Populate data:

```javascript
const course = await Course.find().populate("instructor");
```

Now instructor details appear.

---

# 1️⃣4️⃣ Query Features

Filter:

```javascript
Course.find({ price: { $gt: 100 } });
```

Limit:

```javascript
Course.find().limit(5);
```

Sort:

```javascript
Course.find().sort({ price: -1 });
```

---

# 1️⃣5️⃣ Example Complete Mongoose Code

```javascript
import mongoose from "mongoose";

mongoose.connect("mongodb://127.0.0.1:27017/courses");

const courseSchema = new mongoose.Schema({
  name: String,
  price: Number,
  assignments: Number,
  projects: Number
});

const Course = mongoose.model("Course", courseSchema);

async function run() {

  await Course.create({
    name: "MongoDB Course",
    price: 150,
    assignments: 12,
    projects: 5
  });

  const courses = await Course.find();
  console.log(courses);
}

run();
```

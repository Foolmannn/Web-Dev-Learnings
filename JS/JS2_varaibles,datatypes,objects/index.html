## 1ï¸âƒ£ Variables (rules & methods)

### Ways to declare

```js
var x = 10;    // old, function-scoped, avoid
let y = 20;    // block-scoped, can change
const z = 30;  // block-scoped, cannot reassign
```

### Rules

* `let` / `const` are **block scoped** `{ }`
* `const` **must be initialized**
* `const` â‰  immutable (objects/arrays can still change)
* Use **`const` by default**, `let` when reassignment is needed

## 3ï¸âƒ£ Objects

### Create object

```js
const user = {
  name: "Alex",
  age: 25,
  isAdmin: true
};
```

### Access values

```js
user.name        // dot notation
user["age"]      // bracket notation
```

### Add / update / delete

```js
user.city = "NY";
user.age = 26;
delete user.isAdmin;
```

### Object methods

```js
Object.keys(user);    // ["name", "age", "city"]
Object.values(user); // values
Object.entries(user);// key-value pairs
```
